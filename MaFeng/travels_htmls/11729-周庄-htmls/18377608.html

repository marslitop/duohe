两天玩法
第一天 迷楼(20分钟) → 周庄 博物馆(1小时) → 古戏台(30分钟) → 双桥(20分钟) → 张厅(1小时) → 富安桥(20分钟) → 沈厅(1小时) → 《四季 周庄 》演出(80分钟)
到达古镇先去酒店办理入住，稍作休息。午饭过后开始游览古镇，先去精致小巧的迷楼逛一逛，再去 周庄 博物馆了解一下 周庄 的历史，然后去古戏台看一场正宗的昆剧表演，看完表演去双桥拍拍照发发呆。随后观赏沿河临街的张厅，出来后去富安桥走一走，据说走三遍会事业有成。最后游览沈厅，其中松茂堂最有看点。晚上7：30有《四季 周庄 》的演出，要准时去观看。

第二天 贞丰 桥(20分钟) → 全福讲寺(30分钟) → 南湖秋月园(2小时) → 沈万三故居(1小时) → 古镇水巷游(20分钟) → 三毛茶楼(30分钟)
早晨起来吃过早餐，开始逛景点，从 贞丰 桥开始，到达全福讲寺、南湖秋月园，景点的内部环境非常好。随后可以去沈万三故居，这里还原了 江南 巨富沈万三的生活场景。然后乘船游玩， 周庄 有三条游船线路，蜂蜂们可以自由选择，这里推荐古镇水巷游。最后不要忘了去三毛茶馆，墙上贴了很多三毛的照片，喜欢三毛的蜂蜂不要错过。
贞丰 桥(20分钟) 步行5分钟 全福讲寺(30分钟) 步行4分钟 南湖秋月园(2小时) 步行3分钟 沈万三故居(1小时) 步行16分钟 古镇水巷游(20分钟) 步行14分钟 三毛茶楼(30分钟)
一天玩法
周庄 博物馆(1小时) → 迷楼(20分钟) → 全福讲寺(30分钟) → 南湖秋月园(2小时) → 古镇水巷游(20分钟) → 沈厅(1小时) → 张厅(40分钟) → 双桥(10分钟)
到达古镇，先去博物馆了解一下 周庄 的历史，然后去精致小巧的迷楼逛一逛，随后去全福讲寺、南湖秋月园，景点内部环境非常好。然后选择乘船游玩， 周庄 有三条游船线路，蜂蜂们可以自由选择，这里推荐古镇水巷游。接着去游览沈厅，其中松茂堂最有看点，出来去观赏沿河临街的张厅。最后去双桥拍照。

老婆游记：
11.3（2） 周庄 晴

离开 苏州 博物馆，径直坐公交车向 苏州 汽车站出发。11点多一点就到了，买了去 周庄 的套票， 周庄 门票+车票+船票。一共116元每人，11点40分，准时出发。

温馨提示，当时在 苏州 拙政园门口，有人吆喝，门票只有35，天上没有白掉的馅饼和免费的午餐，其中必有蹊跷。也许会捆绑船票一起卖，绝不会划算，只能被坑。所以建议游客要在正规地方购买门票，出门在外，切勿乱贪小便宜。

坐在车上，望向窗外，不一会离开了 苏州 的老城区，展现在眼前的是一座座高楼林立的现代化国际大都市。 苏州 的大秋裤神采奕奕的矗立在城市的中央，成为了 苏州 标志化建筑。

苏姑娘，再见。

坐了大约一个多小时的大巴，终于抵达 周庄 。 苏州 到 周庄 单算车票16元。不过一般都会推荐给你套票，就是我买的那种116元的。坐上了免费的游览车，向码头驶去。在游船上等了约半小时，1:20游船儿准时开船，浮萍随着碧波荡漾，波光粼粼的河水倒映着远处的宝塔，别有一番情调，此次旅程也算海陆空圆满了。老公指着远处河对岸，那边就是 上海 了。20分钟左右，我们登岸。拱桥下一艘扬着白帆小船，帆上写着 周庄 二字。

从南湖秋月旁边走过，进入 周庄 景区，票据拍照之后，开始寻找我们的春暖花开。春暖花开旅馆就在沈厅后巷弄堂里，刚出检票口就到，非常方便。弄堂头上有一个手艺人，能用草杆编织出活灵活现的大龙，手枪，动物，引人驻足观瞧。

周庄 是一座 江南 小镇，有“ 中国 第一水乡”之美誉。 周庄 镇60%以上的民居仍为明清建筑，仅0.47平方公里的古镇有近百座古典宅院和60多个砖雕门楼， 周庄 民居古风犹存，还保存了14座各具特色的古桥。沈万三故居、富安桥、双桥、沈厅、张厅，怪楼、 周庄 八景不得不逛。

吴侬软语，阿婆茶香，橹声欸乃，昆曲悠远，如一幅“小桥流水人家”的画卷。钥匙形状的双桥，静静地卧在墨绿色的小河上。所谓钥匙，就是成丁字形的河水上建两座桥相互垂直。除此之外还有古老的 贞丰 桥、太平桥……相对年轻的全功桥、福洪桥……随便挑一座都有数百年的历史。古朴动人的桥，不仅见证了明代 周庄 的繁华，水乡人们的喜怒哀乐，也承载着浮躁的都市人渴望返璞归真，远离喧嚣的梦……

送完了行李，才发现奔波了大半天都没有好好吃东西，带着导航，带上老婆和孩子，孩他爸带我们吃奥灶面去了。位置在古牌楼附近，比较好找。或许不是饭点，就我们一桌吃面。团购12一碗 ，比较公道，要了爆鱼面和大排面，面有点稍硬，味道虽然也不错，自然和同 德兴 没法相提并论。沿着古牌楼向前走50米不到发现了一华润万家，价格特别便宜，建议想要补给的同学们出景区找华润万家，明码实价，业内良心，一个大瓶康师傅矿泉水只要2元,酸奶6.5,估计景区里15.千万不要在景区里做待宰羔羊了。
