2019年算不上忙碌，却只去了一次旅行，三月好时节，春回大地，百花盛开，带上妈妈去一次赏花穿越之旅……

无锡水浒城

梅园

无锡 的春天从梅园开始，每年2月 无锡 梅花节开幕，活动50天，园内梅林面积120亩，梅花品种400余种，有梅花1万余株，主要分布在荣氏梅园、古梅奇石圃、凝香道两侧、东横山及北园等五个区域。

无锡梅园景区

👉开放时间：6:00-18:00

👉门票：60元

🚌公交：2路 快3线 40路 52路 53路 67路 72路 83路 87路 88路 89路 106路 112路 131路 158路 310路 766路 啤酒花园专线梅园站。
地铁：地铁2号线梅园 开原 寺站下。

我们去的时候已经是尾期，梅花凋谢得差不多了。

南长街

南长街是 无锡 的一条老街，街区以古运河为轴，由寺、塔、河、街、桥、窑、坊众多景观组成特色环境，构筑了独具风韵的 江南 水弄堂，夜景挺美的，坐在河畔感受微风拂过，别有一番味道！

无锡 三绝之一，银鱼、白虾、白缌鱼号称 太湖 三白，是 无锡 著名系列菜“ 太湖 船菜”的招牌食材，由于其色泽均呈白色，因而称为“ 太湖 三白”。 太湖 三白制作菜肴的方法，多为清蒸白灼，强调保持食材的原始鲜味。

太湖鼋头渚风景区

无锡 的玉祁水芹始种于明代，盛名于清代，与玉爪蟹、乌背鲫鱼并称为玉祁的“三珍”。相传乾隆皇帝两下 江南 ，都钦点玉祁银丝水芹菜，品尝后脱口而吟：“玉者白也，白者盛也，银丝水芹玉祁产也”。

太湖鼋头渚风景区

鼋头渚

👉2019 无锡 鼋头渚樱花节时间是3月1日-5月1日，夜樱开放时间：3月22日-4月7日
👉夜樱亮灯时间： 17:00-21:00，有灯光的主要在 长春 桥和樱花谷。
（2020年的应该也差不多是这个时段吧）

太湖鼋头渚风景区-樱花谷
太湖鼋头渚风景区-樱花谷

👉票价：鼋头渚风景区白天的门票为90 元（含园内公交车、客渡船），夜赏樱花的门票为 80 元（不含游船），白天进可以呆到晚上就不用再买门票。

👉半价政策：
①6周岁(不含) -18周岁(含)未成年人(凭有效身份证件)享半价优惠。② 江苏 省外60周岁(含) ~69周岁(含)的老人(凭身份证)享半价优惠。
③全日制在校大学本科及以下学历学生(凭本人学生证)享半价优惠。

👉鼋头渚樱花开放时间：早樱开花时间大约在3月中旬，今年天气较冷，樱花推迟盛花花期，这两天已经开了不少，月底吉野樱进入全盛时期。晚樱开花时间大约在4月中上旬。

👉樱花最佳观赏点： 长春 桥，樱花谷，赏樱阁，樱花邮局。

👉赏樱路线推荐
1、步行赏樱线：充山大门 → 充山隐秀 → 挹秀桥 → 樱花林 → 赏樱阁 → 人杰苑 → 樱花谷 → 樱缘桥 → 樱谷流芳 → 长春 花漪 → 具境胜境 →十里芳径
2、自行车赏樱线：充山大门 → 充山隐秀 → 十里芳径 → 樱花谷(樱缘桥、樱谷流芳) → 樱花林(人杰苑、赏樱阁) → 长春 花漪 → 具区胜境
3、最高效赏樱路线 ：犊山大门→ 长春 桥→樱花谷(早点入园，越早越好，早上六、七点可以入园了)
