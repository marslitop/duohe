千年古镇枕水人家 不负春光乌镇行

相关目的地：乌镇
去看看> 不再显示

早在唐代，白居易就在诗中写到：“江南好，风景旧曾谙，日出江花红胜火，春来江水绿如蓝。能不忆江南？”，现代人对江南的追捧与热爱与古人如出一辙，每每想到这首诗，思绪总会飘向那风景如画的江南。
春回大地，万物复苏，值此春暖花开之际，更是出游的好时节。对于江南古镇有着迷之向往的我，提前做攻略，春秋是乌镇的最佳旅游季节，果断敲定行程和时间，周末我带上老公，老公带着钱，才算不辜负这大好春光。
住宿作为出行的重中之重，肯定是要提前预订的。在朋友的推荐之下，在木鸟短租上订房。房东人很好，听说我们过来玩，需要租车，直接告诉我们萧山机场有直达景区的班车，1小时一趟很方便，帮我们省去了接机的钱，还告诉我们一些旅游的注意事项等等。无比期待中，终于等到周末。
下了飞机，坐了一个半小时的班车，终于来到了梦中的乌镇。

绿水灰房，交映成画

乌镇古建筑

不失文艺的古镇一角
走进乌镇，白墙黛瓦、小桥流水，仿佛置身水墨画之中；雕梁画栋、石巷老屋，这里有“鱼米之乡，丝绸之府”之称，这里是江南水乡中难得的水阁，这里是中国最后的枕水人家，让人忍不住萌生“来过，便不曾离开”的想法。
到景区买票，多亏了木鸟房东的建议，让我们买联票。东栅100元，西栅120元，东西栅联票150元，如果您是住在景区内，那么联票两日内是有效的，也是最划算的哦。另外，据木鸟房东说，2017年8月1日起东栅景区门票将上调至120元/人，西栅上调为150元/人，所以想去的小伙伴们抓紧了哦。 
在乌镇的第一餐：酱爆螺蛳、东坡肉、白水鱼。先吃饱才有力气看风景~

第一天：到达——东栅
游览路线参考：财神湾→ 江南百床馆→ 江南民俗馆 → 三白酒坊 → 宏源泰染坊 → 江南木雕陈列馆 → 茅盾故居 → 汇源当铺→欣赏夜景
第一天主要游览东栅。按照网上的攻略建议，从财神湾出发一路向西游玩，了解江南民俗，看酒坊和染坊，以及精美的木雕、当铺等等。

在乌镇游玩，一定要住上一晚才算体会到完整的乌镇，因为这里的夜景真的真的真的很美很迷人。这真是相当充实的一天。当夜幕逐渐暗下来，你会看到一个完全不一样的乌镇。

 我们把住宿选在了西栅，西栅景区设施齐全便利，又能欣赏到乌镇最美的夜景，所以住宿优选西栅哦。木鸟短租上定的房子，非常漂亮，是欧式城堡房，我跟老公也非常满意，房东人超级热情，一百分。

第二天：西栅——回程
游览路线参考：安渡坊渡口 → 草木本色染坊 → 叙昌酱园 → 亦昌冶坊 → 喜庆堂 → 昭明书院 → 老邮局→ 水上戏台 → 乌镇桥里桥 → 白莲塔寺 → 益大丝号作坊 → 女红街
攒足精力，第二天继续逛西栅。西栅的景观因人文景观与自然景观的完美结合而深受游客喜欢。如果说东栅适合走走逛逛了解人文历史，顺便买一些纪念品，那么西栅则适合游览散心和拍照。另外，景区内还可以体验摇橹船，近距离看河两岸的风景和建筑，那感觉很美。
一直有人说，现在的乌镇，经过统一的改造，商业化气息渐浓，虽然多了一分精致，却也少了一份历史的沧桑。但是，当你真正置身其中，仍可以感受到浓郁的吴越古镇地方文化，保留也相当完整，并且一直散发着属于她自己的光，吸引诸多游客前往欣赏与膜拜。

在恋恋不舍中，我跟老公踏上了回程的飞机。虽然只有短短的两天，但是足以让我爱上乌镇这个江南水乡，静谧的生活，淳朴的民风，让人的幸福指数直线上升。
乌镇，一个散心首选之地，一个洗涤心灵之地，一个来过就不愿离开的地方，一个水墨画中古镇。不负春光，带上你的家人和朋友一起，来乌镇欣赏水墨画中景。
