十月—漫游乌镇杭州

相关目的地：乌镇

  • 出行天数/3 天
  • 人物/ 和朋友
  • 人均费用/2000RMB
这篇游记豪迈地提到了：鑫兴客栈 ，东栅

此行算是最临时的决定。在9.26号和闺蜜说准备过段时间去看她。她说:择日不如撞日.就十一过来吧正好也休，可是票压根就不好买。最终买到28号的硬座不靠谱的我= =因为涂指甲油晚点，赶去火车站的时候Z31已经弃我而去，一个人坐在地下通道的KFC悔过的时候，一小伙伴赶过来陪我。最终还是买了29号的机票如约去了。
乌镇我也说不清哪里好。吸引我的大概是那白墙黛瓦吧。又或者我向往那悠然自得的生活。
时间2013.9.29—2013.10.3游线：余姚—乌镇—杭州
人物：我和小伙伴
花费：2000（闺蜜有接待一部分）
day1
am6:10  
住宿：陶然居客栈￥138 Add: 嘉兴市乌镇环河路528号（西栅景区）
宏基汽车站搭乘最早的机场大巴￥32去天河机场。am7:00抵达 提前两小时办理登机手续 9:30登机 11:30抵达萧山机场，朋友接到直接去了余姚汇合，直接余姚出发在沧田站买到pm4:10去嘉兴的票，抵达嘉兴的时候最后班去乌镇的专线刚走，只好买票去桐乡，由于时间正好是上下班出行的高峰期，司机师傅车技惊人超了去乌镇的专线车，换成乌镇专线，顺利于pm7:30抵达乌镇汽车站，客栈老板来接。

机票￥644去哪儿网定


恒盛酒家￥70
味道不错。跟老板讲陶然居介绍来的有优惠 

 老板人很好，特意留的房间好公主的，粉色桃花墙壁，单独空调。浴室也很别致浴室门有点小性感啊= = 
 
day2
乌镇东西栅联票￥150单独购买 东栅￥50 西栅￥120
住宿：鑫兴客栈 ￥138  Add：桐乡乌镇环河大道572号（西栅景区北大门）
晚餐：恒盛酒家
想看夜景的话先游东栅再去西栅，门票当天有效，限一次出入。如果住在乌镇西栅里面的话，住宿是统一管理的想出入购物什么的只需在住宿管理处登记便可三天自由出入西栅。景区住宿稍贵量力而行！东西栅一天时间的话我个人觉得会有走马观花的感觉时间有点赶， 

乌镇东栅景区

走进东栅映入眼帘的便是这家茶铺，很诗情画情呢！ 

乌镇东栅景区

乌镇东栅景区

个人觉得东栅比西栅美！逛完东栅不知不觉都下午两点了赶到西栅的时候都下午三点了= =

乌镇东栅景区

西栅有免费和付费的船过河，免费的队永远排的那么长= =步行看到的风景或许更美

仙芋传奇还是比较爱这个抹茶味冰淇淋球￥30

总结：东栅时间耗费过多，导致西栅游玩时间不够，很多景点都关掉了，最可惜的是向阳花开去的时候都晚上八点此行没带单反，我的大片大片的向日葵= =
day3
杭州
住宿：如家快捷  ￥343 Add：杭州市下城区池塘巷7号 ( 武林广场商圈  )
步行至公交站牌1路到汽车站下，乌镇—桐乡￥5—杭州 G7313￥26.5
pm3:00杭州西站乘公交3路到小商品市场下，穿过小商品市场后面就是了。当然你得找到公交站牌，问了好多人找了好久才找到的= =太不容易了晚上7点出发城站接人==搭乘地铁。感受了杭州地铁，打算吃必胜客的，去的时候关门了，走了好久也没找到个能吃饭的地方= =
晚餐：85°C蛋糕还不错  兰州拉面真心不好吃= =一天就这样过去了明天就是残忍的十一了= =

day4
西湖—雷峰塔
住宿：如家￥343
晚餐：外婆家  Add:杭州西站浙江大厦17楼 ￥200
小商品市场对面公交站牌乘5路哪站下忘了，当时就是跟着人走的。跟着人走就走到了西湖边上，你们感受下= =


断桥= =着看的那是景，分明就是人头，国人太恐怖= =看着这么多人决定去大雷音寺！好不容易挤上公交百般心酸的来到了雷峰塔，好不容易买到了票￥40国庆￥20平时


重建的= =失望了

塔顶看风景还是蛮不错的= =

雷峰塔特产有四种，觉得这两种比另两种要好吃！

赶到灵隐寺的时候下午5点了。看着庞大的购票大军，放弃了！！！

超堵两站路走了一个小时抵达外婆家pm8:00取票排队永远都有那么多人在自己喜欢的店子里吃饭点菜根本停不下来= =他妹的让他先上饭，非得让菜上完，菜是上完了你大爷的饭也没了一桌菜怎么吃-（左下角一大张图是2号的，剩下的散图是3号的）

day5
午餐：外婆家 Add：杭州大厦D座店
由于我非要去外婆家吃饭还非要打包面包秀导致赶到城站已经过了检票时间幸而票在最后关头退掉了赶去汽车站买票回余姚收拾行李回武汉= =

总结：这次人头看的太不爽我还会再来的！

时间2014.5.9—2014.5.12
游线：乌镇—西塘
人物：我和小伙伴
方式：自驾游

day1
杭州—乌镇 G592￥285
住宿：西栅景区旁￥160
晚餐：恒盛酒家
PM3:00抵达杭州，约6点抵达乌镇景区，吃饭闲逛睡觉。
day2
乌镇西栅￥120 西塘￥100
住宿：嘉兴安泰商务宾馆 Add: 嘉善晋阳东路8-12号 ( 嘉善县  )
午餐：恒盛酒家旁
PM6:00离开乌镇。≈30min抵达西塘，
要么没房要么太差，最后决定先感受西塘夜生活。
西塘平时查票不严。有客栈代买票￥50.早8前下5后进去不要门票。晚上九点以后外地车可以开进去。停车场一天￥10过夜￥20.背包带相机的大门处保安不让进要查票= =


day3
西塘
晚餐：外婆家 Add:城站浙大17楼
杭州—武汉 Z47￥222.5


还记得去年来的时候还在修建，现如今模样却大变样了。

晚上的乌镇依旧是熙熙攘攘，拍照留念的人永远那么多。我也是其中之一。

还是熟悉的味道同行的人换了又换，
2012年底开始吃素，2013年去的时候没敢点白水鱼她怕一个人吃不掉，今年终于点了虽说我没吃却看着就很有食欲！

停车场很大景色也不错！

还是同一个地方同一个手机，只是拍摄时间的变化。

穿梭其中的人们，却少了你我。 

 凤冠


还是同一家店还是同一碗芋圆，味道却不似记忆里那么好吃了。


爬满一面墙的爬山虎。一盏灯、一点蔷薇、

那天小雨微醺，拾级而上，信步而行。


邻水而居，你是不是同我一样向往？

去年的向日葵今年换成了油菜

月老庙求的就是姻缘？我便都不求 。

蔷薇花开。  

游船

白莲塔登上顶便能看见大运河

登高而望远。京杭大运河隋炀帝的杰作。

白天的水剧场 

露天电影院

乌镇邮局值得一去。

酒吧一条街伍佰黄磊都有在这落户。

闲适雅居 

曲径通幽处

 宠辱不惊，看花开花落，胜似闲庭信步；去留无意，望云卷云舒，相伴晨钟暮鼓。

小桥流水人家

青旅 

 甜、酱油味重，实在是吃不惯！

西塘夜景、

酒吧真的好有特色，这只是其中一家。

文艺小清新。特别的小店我很喜欢。

想了很久的面包秀，它已换了新模样，味道也不似原来那般好了。一如既往的物廉价美。


自驾游用的是导航。
总结：西塘很商业美食很多，双休的西塘人多的不像话。堪比十一盛况。白天人超多没进去在西塘电影院看了《同桌的你》返程，打道回府。
