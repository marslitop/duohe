乌镇底蕴


乌镇 是 江南 四 大名 镇之一

它作为一个具有六千余年悠久历史的古镇

白墙、黛瓦、小桥、流水......

携带着 中国 水墨画的气息扑面而来

雕梁、画栋、石巷、老屋......

见证着历史沧桑的风情画卷映入眼帘

乌镇之美景

■东栅

保留了原生态的建筑，可体验民俗民风；跟团的游客一般都是游览东栅。

【门票】120元/人。

【开放时间】7:00-18:00。

【建议游览时长】3小时。

■西栅

【门票】150元/人。

【开放时间】9:00-22:00。

【建议游览时长】5小时。

200/人

东栅

东栅很小，一个充实的上午轻轻松松可以逛完的。

乌镇 分为东栅、南栅、西栅和北栅。其 中东 栅和西栅为旅游景区，由于旅游发展所需，西栅原居民已几乎全部迁出祖居地，而其他几个地方均有当地居民居住。东栅以旅游观光为主题，能够体验到原生态的住宅和文化；西栅以商务旅游和休闲为主，是比较成熟的商业旅游区，开发完善，吃喝玩都很方便；南栅和北栅尚未完全开发，比较原生态，目前免费；东西栅游览均收费。

东栅

东栅很小，一个充实的上午轻轻松松可以逛完的。

宏源泰染坊，始于宋代，东栅蓝印花布制作基地，是必游景点之一。 蓝色的布匹随风飘动，十分唯美，上面的印花纹理别有韵味。

公生糟坊，这里是当地酒文化的集中展示地，酿酒的全流程都会呈现给游客。

乌镇 最知名的美食莫过于红烧羊肉，鲜嫩多汁、丰腴肥香。

翰林第的皮影戏，每天有很多场次，中午会休息一小时。

酒坊里有卖冰镇醪糟，烈日下最好的解暑神器。

木雕馆陈列的 江南 地区工艺品，也都是生活用具。

茅盾故居，也叫茅盾纪念馆，人文气息浓厚的地方。

抵达西栅后的第一个景点，应该都是草木本色染坊，大小规模与宏源染色坊不相上下，布匹的颜色还多了红色和黄色等。

西栅也零星分布着关于当地风土人情介绍的展馆，如图上的 江南 民俗馆，规模较大的还有三寸金莲馆。

昭明书院，闹市中的安静天堂，主厅保留了图书馆的功能，里面的游客静悄悄，如需进入，必须按照规定将自己的随身物品寄存。

西栅景区后半段，随便河畔走走，都能看到小桥、流水、乌篷船等 江南 经典元素。

西栅景区南北两岸中有很多小石桥，随便上去走走看看，都是满目风景。

乌镇路线推荐

■ 乌镇 精华1日玩法

如果一天内既要玩东栅又要玩西栅，那么上午就要早点到 乌镇 ，先用2-3小时游览东栅，午饭后开始在西栅游览，一直玩到晚上在西栅看夜景。

D1： 江南 百床馆(30分钟) → 江南 民俗馆(20分钟) → 宏源泰染坊(30分钟) → 江南 木雕陈列馆(0.5-1小时) → 茅盾故居(30分钟) → 草木本色染坊(30分钟) → 昭明书院(30分钟) → 老邮局(30分钟) → 白莲塔寺(30分钟) → 女红街(30分钟)

乌镇经典路线2

■ 乌镇 2日玩法

乌镇 分东栅、西栅、南栅和北栅。东栅和西栅虽然已商业化，但西栅的夜景非常漂亮。南栅和北栅是生活区，满溢着市井气息和纯粹的古镇风貌。

D1：财神湾(30分钟) → 江南 百床馆(30分钟) → 江南 民俗馆(20分钟) → 三白酒坊(30分钟) → 宏源泰染坊(30分钟) → 江南 木雕陈列馆(0.5-1小时) → 茅盾故居(30分钟) → 汇源当铺(15分钟)

D2：安渡坊渡口(10分钟) → 草木本色染坊(30分钟) → 叙昌酱园(20分钟) → 亦昌冶坊(20分钟) → 喜庆堂(20分钟) → 昭明书院(30分钟) → 老邮局(30分钟)→ 水上戏台(30分钟) → 乌镇 桥里桥(10分钟) → 白莲塔寺(30分钟) → 益大丝号作坊(20分钟) → 女红街(30分钟)

☆Tips：

在 乌镇 游玩，一般只有住上一夜才算体会到完整的 乌镇 。东栅景区大概3小时，西栅5-7小时差不多就够了。

乌镇之住宿

乌镇 住宿

西栅景区：西栅的住宿条件非常发达，从高档会所到临水客栈应有尽有。其中最受推崇的是当地民宿，由景区统一管理，房子都是当地人的民宅，很有特色，管理也很好。住宿在西栅景区内，可以办理西栅景区临时出入证。

东栅景区：东栅虽然没有西栅那么繁华热闹，但是保留了很多当地原始的风情和韵味。价格要比西栅便宜，所以住在东栅也是不错的选择，但需要注意的是，东栅每天早上8点要清场，如果你住在东栅景区内，最好8点以后出来。

东西栅景区门口：附近也都还有很多的家庭旅馆和宾馆可供选择，价格比景区内便宜很多，交通也很方便，适合追求高性价比出游的朋友们。

酒店：世界遗产级古镇与五星酒店配置的完美结合，推荐西栅景区枕水度假酒店，标间830元起。

会所： 成功 商务人士的雅集之地，叱咤商场之余的禅意之园，推荐 乌镇 锦堂会所，高级大床房，1500元起。

客栈：推窗临河，枕水而居，归家的感受，享不尽惬意与浪漫，推荐 嘉兴 月河客栈、 乌镇 万宝座客栈、两府客栈、通安客栈，100元至1000元不等。

民宿： 江南 风情水乡人家，体验星级宾馆洁净与水上人家亲情，推荐 乌镇 民居计阿姨、殷阿姨家等，民宿多在400元左右 。

乌镇之交通

乌镇 交通

飞机：目前只有 杭州 萧山 机场有直达 乌镇 的班车，其他地方都需要下机后周转汽车或火车前往 乌镇 。每天有多趟快客班车往返 乌镇 （ 桐乡 ）和 杭州 萧山 国际机场，票价约为46元/人。若从 萧山 机场直接打车去 乌镇 ，费用在300元左右。

汽车：目前， 上海 、 杭州 、 南京 、 温州 、 绍兴 、 苏州 已开通直达 乌镇 的旅游专线，游客可到各旅游集散中心乘坐。到 乌镇 的一般路线是目的地—— 桐乡 —— 乌镇 或者目的地—— 嘉兴 —— 乌镇 。其中，汽车是最方便的到达方式。

火车：乌 镇没有火车站，前往 乌镇 旅游，坐火车可到 乌镇 周边地区之后再到附近汽车站进行中转。到达 桐乡 火车站后，可直接换成k282路公交车，直接抵达 乌镇 汽车站， 桐乡 火车站发往 乌镇 的公交首末班时间为7:40和17:50，每隔30分钟一趟，票价5元，用时在半小时左右。出租车前往80元起。

tips： 乌镇 虽是 浙江 桐乡 市的一个镇，但去 乌镇 一般先到 上海 、 苏州 或 杭州 ，然后再坐汽车去 乌镇 。

自驾： 乌镇 距 桐乡 市区13公里，距周围 嘉兴 、 湖州 、吴江三市分别为27公里、45公里和60公里，距 杭州 、 苏州 均为80公里，距 上海 140公里。自驾都是不错的选择哦！

A. 杭州 自驾线路：全程约80公里，行程时间约1小时40分钟；

乌镇之注意事项

1、请避免夏季正午时去 乌镇 游览，白晃晃的 日光 直直地晒下来，也没什么廊棚可遮挡，让人感觉非常不好。

2、 乌镇 东栅景区与西栅景区之间有免费班车接送，营运时间通常为7:30至17:20。周一至周四的班车间隔时间为20分钟，周五至周日的班车间隔时间为10分钟。

3、最好避开节假日，因为这段时间不仅住宿等价格很贵，而且人多就无法品位古镇的宁静美。

4、去过的游客感觉东栅的景点一般半天就可以逛完了，而西栅内容比较丰富些，可以花时间在西栅。

1. 乌镇 总共分东栅、西栅、南栅和北栅。目前开放的是东栅和西栅，南栅和北栅是三轮车夫所指的“老 乌镇 ”。

2. 当地规定，镇上居民不准私自营业，所以像住在居民家、吃在居民家、在居民家买工艺品，都是不允许的，一般都是一早一晚进行，或偷偷进行的。

3. 门票当天有效，时间从上午8：00到下午17：30（冬季17：00）。双休日和黄金周期间从晚上18：00—21：00进入景区需要购买夜票。

4. 每天早上7：00—8：00，都会有保安巡街清场，把所有街上的游客都请出去买票。同样在17：30—18：00之间也是如此，要再买夜票进入。

5. 在 乌镇 可以雇黄包车夫作导游，车夫大多是本地人，兼有导游身份，而且他们还认识许多私人景点，会带给你意外收获，一辆黄包车大约15元。

6. 每年3月底～4月底 乌镇 会举行香市（ 江南 水乡狂欢节），茅盾笔下的“香市”场面会再此重现。不过在“香市”期间游客比较多，喜欢清幽的游客可以避开这个时段。
