前言:本次 乌镇 游的落实
农历新年之后大家开始上班，在办公室讨论起要一起自费组织一次短途游玩。大家都是欣然同意，大概在3月份，经过了讨论之后，大家一致把目的地定在了 http://www.mafengwo.cn/travel-scenic-spot/mafengwo/14575.html ( 浙江 ) 省 http://www.mafengwo.cn/travel-scenic-spot/mafengwo/64696.html ( 桐乡 ) 市的 http://www.mafengwo.cn/travel-scenic-spot/mafengwo/10434.html ( 乌镇 ) 。之后，我就迅速的预定了酒店。后顾之忧先解决好 开车路线和单程用时
从我们 http://www.mafengwo.cn/travel-scenic-spot/mafengwo/10207.html ( 苏州 ) 过去应该说是挺近的。路线的话，分为走高速和不走高速2种。
走高速路线:
常台高速-沪渝高速-苏震桃公路- http://www.mafengwo.cn/travel-scenic-spot/mafengwo/10434.html ( 乌镇 ) 。全程85km左右，大概需要85分钟。

不走高速路线:
友新快速路-230省道-苏震桃公路- http://www.mafengwo.cn/travel-scenic-spot/mafengwo/10434.html ( 乌镇 ) 。全程82km左右，大概需要100分钟。
==========================================================
因为距离比较近，用时差别不大，走高速的话，大概需要过路费30元。
4月21号中午前:到达 乌镇
旅游的日子定下来之后，大家都是满怀期待。但是有一件东西却是大家都担心的，不可控制的。是什么？钱不够？不是的，当然是天气。感谢上天帮忙，每次我们出行都是好天气，就算天气预报说了要下雨，照样是晴朗的好天气，也许是我们的运气比较好吧。
中午到达后我们办理了入住手续之后，吃好午饭我们就向 http://www.mafengwo.cn/travel-scenic-spot/mafengwo/10434.html ( 乌镇 ) 西栅景区前进！
我们入住的那年晚村精品度假酒店，距离西栅景区大概800m,步行到入口大概是7-8分钟。
西栅-我们来啦。在进入景区之后，我们也像其他的游客一样，把整个景区逛了个遍。
