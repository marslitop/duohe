五一小长假三天如何玩转 乌镇 ，这份实用贴请收好
| | 行程安排
第一天：东栅
第二天：西栅
第三天：乌村+ 乌镇 互联网国际会展中心
| | 相关价格信息
东栅：100元/人
西栅：120元/人
东西栅联票：150元/人
乌村休闲套餐：280元/人（成人）、220元/人（儿童）
乌村住宿套餐：780元/人/晚起
乌村+ 乌镇 互联网国际会展中心游览套餐：100元/人起
| | 住宿
乌镇 西栅景区现有15家酒店，包括民宿、精品酒店、行馆、高档酒店等不同类型。此外，乌村现有 桃园 、磨坊、酒巷、渔家、米仓、竹屋、知青年代七个各具特色的组团。西栅与乌村，可以满足你不同的需求。
若想体验 乌镇 人家可以选择民宿、通安客栈，适合闺蜜朋友；若是喜欢庭院深深的大家风格，可以选择宜园、盛庭等行馆酒店；若是喜欢田园闲趣，乌村是家庭度假的好去处；若是学生，可以选择紫藤青年旅馆、水巷青年旅馆，比较经济实惠。

13:00 抵达东栅景区

16:30-18:00 晚饭
18:00-22:00 夜游西栅
13:00 抵达东栅
鉴于小长假第一天，可以睡个晚觉，下午抵达东栅景区，将行李寄存东栅免费寄存处后。如你网上已购票，可凭短信显示的二维码扫码进景区。
13:00-16:00 游览东栅
东栅2001年正式对外开放，属于观光型景区，民俗、名人场馆多，是挖掘 乌镇 传统文化的必玩之地。、
推荐游览路线：东栅游客服务中心—财神湾—香山堂—财神堂—白床馆—三百酒坊—蓝印花布坊（宏源泰染坊）—乌青水龙会—拳船—晴耕雨读— 江南 木雕馆—余榴梁钱币馆—茅盾故居—修真观—翰林第（皮影戏）—汇源当铺

财神湾高杆船表演
一天两场，早上10:30，下午2:30，表演时间为10分钟，特殊天气不表演。

Tip：修真观、翰林第皮影戏、汇源当铺三个景点都在外面，由于门票一次有效，所以最好放到最后游览。参观木心故居纪念馆需在木心故居纪念馆官网（ http://www.wzmuxin.com/ ）提前预约。

16:00-16:20 抵达西栅
经过一个下午的东栅之行后接下来乘坐免费班车前往西栅景区，注意哦，东栅至西栅最后一班免费班车为17:35。
16:20-16:30 办理入住
抵达西栅景区之后，前往大前台办理入住，行李可以直接托运至入住酒店，这样可以轻松愉快的夜游西栅了。
凭二维码进入景区时，可以在检票处领取一份纸质地图。西栅景区较大，除了步行游览，可以在景区入口码头乘坐免费大船进入，直接抵达安渡坊码头，或者乘坐游览车抵达目的地。

16:30-18:00 晚餐
除了景区内酒店、民宿的美食之外， 乌镇 还有多家用餐之地，天 南海 北菜系缤纷，酸甜咸辣风味交融。
辣 乌镇 的香辣，民俗的朴实，后街酒家的风味，锦岸私房菜的秘制、淡茶饭的“粗茶淡饭”、京杭河鲜馆的河鲜……景区内总有一家的美食，能满足你每一个蠢蠢欲动的味蕾。

18:00-22:00 夜游西栅
一曲似水年华，一阕温婉多情。西栅的夜景是 乌镇 行的必玩项目。

8:00-8:30 晨起散步西栅
8:30-9:00 早餐

12:00 退房
12:00-13:30 午餐
13:30-17:00 自由游览西栅
17:00-18:00 乌村晚餐
18:00 自由活动
入住景区的客人，可以直接凭房卡在酒店享用早餐。入住民宿的客人可提前将想吃的早餐告诉房东，第二天就可以享用定制早餐了，是不是很贴心。
吃完早餐，趁着景区还未开园，在小巷里、大街上摆pose多拍几张美美的照片吧。
9:00-12:00 游览西栅
推荐游览路线：草木本色染坊—叙昌酱园—亦昌冶坊— 水上 集市—万兴桥—喜庆堂—放生桥—灵水居（王会悟纪念馆、茅盾纪念堂、孔另境纪念馆）—定升桥—昭明书院—三寸金莲馆—徐昌酩画院—乌青水龙会— 乌镇 老邮局—乌陶坊—国乐戏院（评书场）— 水上 戏台—恒益堂药店— 肇庆 堂银楼—桥里桥—白莲塔（白莲寺）—童玩馆—关帝庙—水剧场—益大丝号—乌将军庙—女红街
若是想了解更多的 乌镇 历史，可以在游客服务中心导游咨询处邀请一位导游全程陪同讲解，大概3小时左右可以游览玩西栅相关景点。当然，如果想简单随意点，可以租赁语音导游机边走边听。
Tip：西栅语音导游机30元/台

17:00-18:00 乌村晚餐
晚餐可以在乌村知青餐厅吃，新鲜的一小时蔬菜，新鲜美味。
| | 第三日行程
8:00-8:30 早餐
8:30-12:00 游玩乌村
12:00 退房
12:00-13:30 野火饭
13:30-15:00 游览 乌镇 互联网国际会展中心
15:00 返程
早餐之后，可以询问一下乌村CCO，为你量身定制一个乌村游计划。
乌村于2016年2月正式对外开放，采用一价全包式的概念，一键遇到吃住游，包含三餐，几十项免费活动和乌村各类小吃美食。乌村的活动多样，每个项目都有我们CCO陪同教授玩法，钓鱼、射箭、编织都是不错的体验。
| 不可错过的乌村活动 |
骑行

13:30-15:00 游览会展中心
凡是预订乌村休闲套餐、乌村住宿套餐以及景区部分酒店客房，可免费参观 乌镇 互联网国际会展中心，领略大咖同款会议展厅的风采。

15:00 返程
三日行程圆满结束，小长假最后一天，打道回府。
