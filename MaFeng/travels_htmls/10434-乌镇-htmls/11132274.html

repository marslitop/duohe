#心的乌镇，思君许久#我在乌镇的三天两夜。

去看看> 不再显示
  • 出发时间/2018-06-16
  • 出行天数/3 天
  • 人物/ 和朋友
  • 人均费用/1900RMB

乌镇 ，一个念了许久的地方。因为怕极了过度的开发以及过分的商业化，这几年与 乌镇 错过了一次又一次，但来了之后才知道，商业化也好，缺少原生态也罢，所有的一切，通通抵挡不了它的魅力。
这一次的 乌镇 之旅，来的也是突然。原先端午假期的计划，并未曾考虑过往华东这一带来，只是想着利用三天的假期去一趟 福建 霞浦 ，来一次短途的自驾游兼跟着大师学摄影之旅。 然而最后的最后，便演变成了我和珠姐的临时“叛逃”，来到了这心念许久的地方。

@乌镇，怎么去？

从 潮汕 出发，有多种方式可以到达 乌镇 。
1. 上海 中转： 潮汕 → → 上海 → / → 乌镇
2. 杭州 中转： 潮汕 → → 杭州 → / → 乌镇
3. 动车直达： 潮汕 → → 桐乡 → 乌镇
高铁和 杭州 中转，班次不多，要么一票难求，要么时间点不对。考虑再三，我们还是选择去 上海 中转。这里需要赞一下江浙沪这四通八达的旅游网，基本你只要到达其中一个地方，都可以或是动车或是旅游大巴的方式中转到其他的城市去，便捷度比之 汕头 去邻居 潮州 家要方便不知道多少倍。
这次去 乌镇 ，因为从 潮汕 飞过来是去的浦东机场，所以我们没有选择动车前往 桐乡 再转，而是在网上订了旅游专线的大巴车票，直达 乌镇 西栅大门， 其实反而更方便的说。

@乌镇，住哪里？

【西栅民宿】
西栅景区内的住宿有多种选择，度假酒店、精品酒店、民宿客栈、青年旅舍，每一种的价格相较于外面都是价格不菲，节假日尤胜。但如果还有机会再来 乌镇 ，我会继续选择在西栅入住。
因为统一管理，西栅的住宿环境舒适度、整洁度和安全性一整个都得到了很大的提升。游客在办理入住之后，景区会统一将行李送至房间， 这样就可以轻装上阵直接开始游玩。民宿虽然是由旧房子改造，但房间的隔音还是很好， 在西栅的两晚，我基本都是倒头就睡一觉到天亮。
至于选择临水与否，两个晚上我们住的一间临水房一间书院房，我觉得是没差的，相比我更喜欢在书院这边的，民宿的空间更大。 在白莲塔的时候遇到一对年轻夫妇，那个先生说了一句也是蛮实在的，天一黑眼一闭，临水什么也就都看不见了。
如果有考虑在西栅景区内入住的话，我的建议是进官网买套餐，包含了住宿、门票、包船游和傍晚登塔等项目，虽然还是贵，但相较而言蛮划算的。

@乌镇，吃什么？

【茅老太臭豆腐】
地点： 乌镇 西栅
消费花费 ：10元/份
推荐指数：☆☆☆
一家将“臭名远扬”的牌匾高高挂在自家店C位的臭豆腐店，在西栅水上集市旁边，只可堂食，不能外带。臭豆腐分为脆皮和不脆皮的两种（ 个人觉得脆皮的更好吃。），一般没有特别交代，老板都会按拼盘 来安 排。酱料自取，有甜酱和辣椒酱。臭豆腐的口感跟平常所吃的不大一样，而且是臭到豆腐里头去， 以致于我刚开始吃的时候还真不怎么习惯。

茅老太臭豆腐

【锦记糕点铺】
地点： 乌镇 西栅
消费 ：定胜糕3元/个，红糖发糕、乌米糕5元/个
推荐指数：☆☆☆☆☆
锦记糕点铺在西栅大街黄金地段，又是 乌镇 旅游重点推荐的美食店，每天在这里都要排着相当长的队伍， 当然也有人少点的时候，那就是定胜糕卖完的那段时间， 如果想要少排队又能吃到定胜糕，那就要请早了。 这里必须强烈推荐定胜糕，真的是好吃也不贵！糯糯的，里面红豆馅甜而不腻。

锦记糕点铺
锦记糕点铺

【民宿美食】
地点： 乌镇 西栅各民宿
消费 ：统一标价
推荐指数：☆☆☆☆☆
西栅中也有其他的一些餐饮店，但每一家都排着超长的队伍，而且价格也相对不那么亲民。 所以在西栅的三天，我们基本都在民宿中解决温饱问题。西栅的民宿餐饮也在景区的统一管理之中，统一的食物供给，统一的菜谱和价格，价格基本跟景区外吃的无异， 这必须对西栅的管理点个大大的赞。再说回各家民宿的口味，网上也有推荐一些网红民宿，但像我们这种吃的经常不对点的也很难排上队，所以我们也就在住的那两家民宿吃（61乙和16丙）， 味道其实都还不错的，而且一家临水，一家在书院旁边很是安静。

【老古树餐厅】
地点： 乌镇 甘泉西路（景区外）
消费 ：150元
推荐指数：☆☆☆☆☆
最后一天的午餐来到老古树其实也是个意外，原本打算简单用个KFC结束 乌镇 的旅游，但在东栅走的晕乎乎的我们一时没有找到，最后才决定打个滴滴到大众点评网上排行靠前的老古树解决午餐，确实没让人失望，味道是很不错的。 酱鸭大概是有上锅蒸过，甜甜的，肉不会太硬。最推荐的是秋葵金针菇，加了蒜蓉和肉沫，美味！

@乌镇，怎么玩？

端午假期的 乌镇 ，无论是西栅还是东栅，都逃不掉的人从众，几条大街上基本塞满了人，几乎没有一点让人停留下来拍照的空间，当然，在那样的情况下，大概也是提不起兴趣拍照的 ，以致于最后一个上午在东栅那里，我们基本就是跟着人流走完整个过程，不用2个钟就走了出来 。
虽说如此，但对于我们这种行程说改就改，纯粹为了拍（zhuang）拍（zhuang）照（B），压根没想一定要去哪个景点打卡的倒是没有大多影响。于是在西栅，我们的攻略是，早出晚归中午休息下，从外面包抄进里面，哪里人少就往哪里钻，哪里好看就在哪里拍照，拍得不够尽兴的第二天继续 ，行程安排什么的，从来都是说改就改，反正最后赶得上回家的飞机就好 。

西栅

网上曾看人说， 乌镇 的西栅商业化太严重，少了太多原生态的味道，走在这里就是在一个全新的景区，没有太多古镇的感觉。 太多这样的评论，让肖想 乌镇 许久的我总是望而却步，离最近的一次，我却是选择了 西塘 ，却也远没有这次在西栅给我带来的触动来得深。

西栅
西栅

我想，或许正是由于这常被人诟病的统一管理，才让西栅有如今这般的干净透彻，一切都是有序，即便节假日你甚至也看不到它过分的哄抬物价， 当然可能跟它本来住的就不便宜也是有关的。但不可否认的，少了时刻提防着被宰的担忧，在这里我们玩的也更开心了些。

西栅

没什么人的小巷子是我们最喜欢的，也是我们心中认为的最有古镇韵味的地方。 干净而又残旧，木门后还顽强生长着一撮撮的墙头草。

清晨的西栅，少了熙熙攘攘的游客，静谧、美好。偶尔路过一两个人，除了匆匆赶往岗位的景区工作人员，也便是我们这些，趁着晨起无人出来拍照的了。 路上遇到一个大概随游客进来跟拍的大哥，看着我们同是捧着相机出来晃荡的，也忍不住给我们指导一二， 比如 哪里光线好些啥的……

虽说不去网红景点打卡，但毕竟这 乌镇 “大饼”就在我们住的民宿对面，一大早趁着没啥人还是要拍拍的。

西栅

在 乌镇 ，有机会一定是要去体验一次游船的。绿水轻舟，摇摇曳曳，看一遍小桥人家，听一些船家故事，给整个行程增色了不少。我们遇到的大叔也是极健谈的，讲了他小时候的 乌镇 ，讲了他在这里工作的一些事情，也讲了他家里孩子的一些事……言语之中，感觉得到的是那份自然流露的幸福之感，工作虽累，但不苦。
