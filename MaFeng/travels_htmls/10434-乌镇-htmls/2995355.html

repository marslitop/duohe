乌镇——水村渔市，一缕孤烟细

相关目的地：乌镇
去看看> 不再显示
这篇游记豪迈地提到了：乌镇民宿，西栅，东栅，老邮局，宏源泰染坊，逢源双桥，白莲塔，昭明书院，喜庆堂，高杆船财神湾元宝湖

对乌镇的向往从很早便开始了。我是一个喜欢古镇的文艺小清新，反反复复乐此不疲。​恰逢亲爱的生日，两个女生便兴奋的一起准备结伴自由旅行。
西栅
​      从杭州辗转到乌镇，天空也转晴了，旅途的疲惫刹那烟消云散。从西栅入口进去后，我便坐着摇摇晃晃的乌篷渡过元宝湖。西栅元宝湖与东柵财神湾相对，取名相映成趣，表达了古镇先民对富庶生活的祈求。一进入乌镇景区,仿佛来到了隔世百年的过去，沿河而建的明清风格民居清秀典雅,风韵独特，青砖灰瓦绿水中，窗边灼灼的野花藤蔓绝对是最美的点缀，经过一个一个的窗口，就如经过一个一个的画框，看着这些美丽的景致，也许不经意间打开的窗户，就会探出一个丁香般的江南姑娘。

元宝湖

元宝湖




上了岸，沿着青石板的小巷，找到之前在网上预定的那家民宿客栈。房东先生说话声音细细的，很热情的招呼着我们。房东先生把房间号给我们之后，帮我们把行李拿到房里。房里的茶几上摆放着两碗房东准备的锅巴茶，他说这也是这里的风俗。他给我们菜单，笑着问问我们明天早餐想吃什么，免费的。房间的陈设古色古香，透着温馨，透着淡雅别致，小到梳子都透着古韵。推开雕花小木窗，就能看到对面的屋檐和小街的青石路。


                                                                     花灯铺



在我住的客栈不远处是一家花灯铺。走近这名为“张恒兴”的花灯铺，门口一盏巨型走马灯吸引行人的眼球。进入店铺，形态各异、琳琅满目的花灯把整个店堂悬挂得满满当当，球型灯、直筒灯、五角灯、蛋形灯、宝塔灯等等，都让人眼花缭乱、目不暇接，使整个店铺都洋溢着融融暖意。乌镇民间另有一个传统习俗，那就是：“提灯走桥”和“花灯长街宴”。每当春节、七夕、中秋、清明，香市等重大喜庆节假日，当地老百姓总喜欢携家人或友人，提着各种各样的花灯，沿着形态各异的古桥，走他个方方正正、团团圆圆。而在那些欢快喜庆的长街宴上，更是要悬挂许许多多五花八门、精彩纷呈的花灯，霎时间，整个长街宴上是姹紫嫣红，煞是好看。


{喜庆堂 }

来到西栅内的喜庆堂。门外门楣上雕刻的是双喜双凤，喜窗点缀。还有门当和朱漆红门上的和合童子像，都洋溢着老宅子的不凡气势和浓浓喜气。有时候我会兀自去思考古代的婚姻，全凭父母之命媒妁之言，有的甚至在洞房花烛时双方彼此第一次见面。这样的情况在现下看来是多么不可思议，可古时却稀松平常。尽管如此，我却不能武断地认定他们不幸福。即便是恋爱自由的现在，无法相守到老的婚姻比比皆是。是古人单纯了还是今人复杂了？或许 许多人相爱了，就想着相守；相守了，就想着白头到老。在内心深处，所深深向往的，即是那四个字——举案齐眉。映着西栅的天光，想到周董的一句歌词：“天青色等烟雨，而我在等你。”

囍庆堂

囍庆堂

                                                                     乌镇文武兼备，德才兼重。一座关帝圣殿，，就是千万乡民的道德楷模和精神寄托；一块青石古碑，就是一个感天动地的忠义教案。关帝庙旁就是飞檐临波的文昌阁  。旧时每逢科考，读书人乘坐小船前来，小船就泊在阁下的河埠边，学子登阁祈告，求金榜题名。文昌阁上供奉着孔子和文昌帝君。建文昌阁和关帝庙体现乡民是尊孔尚儒，倡导忠勇正义。  昭明书院得名于曾在乌镇读书的南朝太子萧统，展现了南朝昭明太子幼年在乌镇从师苦读的佳话。书院环境清幽，石砌的案台静静的在亭子里，仿佛还能听到穿越千年而来的朗朗读书声。一本本泛黄的古书里记载着最灿烂的文化，被安放在这僻静的书院里日复一日、年复一年，带给了这片水乡更多的水墨灵气。。 古人云：“东南之美”。东南地方人杰地灵，滋润出了不少文人雅士，民族英雄。乌镇是历史文化名镇.人文荟萃,钟灵毓秀。唐宋共有十相到此巡视或在此隐居．参观了孔子后代现代作家孔令镜先生府邸，拜谒了一代文豪茅盾故居，领略了当地居民的民风民俗。历史典故千年传颂,吴越遗风历代传承，民族文化感人至深。悠悠岁月，经历了千年沧桑，乌镇清纯,祥和,典雅的古镇风貌,使人深深感悟到了她厚重的历史。

乌镇西栅景区-昭明书院

                                                                  { 花生糕}在沈记花生糕铺里买了花生糕和芝麻糕，花生糕最香。是和着糯米粉做的，店门口还有一个大木槌，老板会让客人自己打打看，有点打糕的意思。包装是那种牛皮纸，外面再系上麻绳。提着这样一包朴素的小糕点，慢慢行走穿梭在乌镇中，好像渐渐走回从前的时光。

                                                                  
      河面上不时地有乌篷船穿梭来去。小船摇曳的终点就是这座双桥，在单独每个桥的桥洞里都可以看到另外一座桥的倩影，如同井中观月，因而博得“桥里桥”的美称。若站南河岸观看，两桥半圆形的桥孔倒映水面，似圆非圆，虚虚实实。桥缝中野树虬枝横斜，站在桥头四望，水阁风光一揽无遗：古朴的石桥或圆或弧,高低错落,风格各异.亭台楼阁错落有致，京杭运河蜿蜒北去，文昌阁风姿绰约，白莲塔巍峨高耸，堪称桥景一绝。“桥里桥”是乌镇最美的古桥风景。
     “你站在桥上看风景，看风景的人在楼上看你，明月装饰了你的窗子，你装饰了别人的梦。”  桥，是江南水乡古镇不可缺少的。乌镇现存30多座桥。因为这里地处杭嘉湖平原，一眼望川，桥成为了游人绝佳的观景台，站在桥上看着来往的小船，依水鳞次栉比的人家，听着乡间小调，感受着"水村渔市，一缕孤烟细"的水乡情怀.{老邮局 }
不知不觉来到乌镇老邮局 。老邮局的建筑风格和西栅大街的木结构的老房子完全不一样，青砖瓦结构，一扇西式的大铁门，整个建筑有点中西合璧的感觉。这个从清光绪年间开办的邮局，如今仍在按照旧时的通邮方式。我养成了一个习惯，每去一个地方旅行，一定会挑选几张美丽的明信片，而后找一家安静的角落坐下来，寄一封给自己，寄一封给此时此刻心里想念的朋友，给在远方给想念的人寄去惊喜和祝福，分享旅途风景，诉说这一刻是如何心动得不能自已。

乌镇西栅景区-中国邮政(乌镇邮局)


                                                            

西栅景区边上白莲寺塔，是西栅的最高处，只是很遗憾开放了三层，如果可以到达七层宝塔的顶端，是不是可以看到身边京杭大运河的尽头呢？乌镇因为京杭大运河而兴盛，这里距离人间天堂的杭州也不过几十公里之遥，可以想见那个以漕运为主的年间里，
乌镇会是怎样的繁华？如今繁华逝去，却又因为曾经的繁华而带来如潮的游人，又何尝不是运河的功绩呢？

乌镇西栅景区-白莲塔

                                                              

夜幕初临，浪漫。暗黄的微弱光线。在黑暗里照亮一个路人的心。
乌镇的浪漫夜景，的确美不胜收。只是手头设备有限无法完整展示，真是有点遗憾。曾听许多人说起乌镇那浓郁的商业气息，白莲塔周边的这一片，聚集了好几间酒吧。而对于年轻人而言，这样的酒吧文化还是喜欢且欢迎的吧。
夜，也赋予了乌镇诗意的时刻。华灯初上，星星点点的灯光亮起来，天色渐暗，越发漂亮，时不时河上有花灯船漂过，很是浪漫。每一处小景，每一个拐角，都满怀意境地可以写句诗出来。始终认为，古镇，是适合情侣们牵手漫步的地方，它本就带着一种温婉柔美的气质而存在，不经意间渲染出的浪漫，一丝丝沁人心脾。

一对情侣说，去什么样的地方、看什么样的风景都不重要，重要的是一起看风景的人。
这句话浅白，却再正确不过。
闲逛在街道上，一些情侣，一些家人同行，一群老友，几个闺蜜，酒吧街几首唱响的旋律，让你觉得那样的熟悉，简简单单的歌词就不经意的唱进了你的心里。小船从这边划到那边，或者从小桥这头漫步到那头，都可以欣赏夜晚小镇不同的景色，感受着浪漫的情调。在城市中，浮躁的心，就这样慢慢地放松了下来。



{ 小巷}
枕水江南梦里的小桥氤氲了千年的烟雨，走下了桥头就来到了江南的巷子里。巷子里历经了百年的石板路，又走过了多少丁香般的姑娘。今天小桥幽巷依旧在那里静静的等待下一场细雨，下一回邂逅。



         河边戏台上咿咿呀呀唱着地道的昆曲。台上也常常有苏州评弹艺人怀抱琵琶三弦，身穿长衫旗袍，风韵优雅，说唱吴侬软语，轻柔绵软之间风显江南水乡之柔美。一曲终了，大家都拍手叫好。

 
                                                                [露天影院}

       大戏院外有一个露天影院。大概是70年代才有的老式放映机。大大的白幕上放着黑白影片。
我突然想起曾经听妈妈说过她小时候看电影的情景。那时妈妈沉浸在童年的回忆里，我也听得津津有味。“时间都去哪儿了？——门前老树长新芽，院里枯木又开花。。记忆中的小脚丫，肉嘟嘟的小嘴巴 。 ”   时光啊，我们来玩躲猫猫，让我不要太快长大好不好？我给你最心爱的糖果，请你不要拿去我的似水年华好不好？
夜深，灯火阑珊。我们回客栈休息，路边偶尔还有三两路人匆匆的欢声笑语，但小巷很快又恢复了寂静。
东柵
                                                                    {清晨]
第二天，清晨，静谧。咯吱一声推开小木窗，天还灰蒙蒙的微亮，呼吸着清新的空气，空气中有木头和泥土青草清香，那里的清晨可以静的只听到清脆的鸟鸣。。我特意起个大早，想去听听东栅早上的开门声，去闻闻江南人家早上的烟火味，才能感受到最真实的乌镇。
下了阁楼，客栈的木板门都还没开，房东先生在厨房忙碌，见我们下来对我们打了招呼。房东先生亲自为我们准备了美味的特色早餐。最爱吃的梅花糕，刚出炉的模样，冒着腾腾热气，淡淡的味道，只有抹了糖的地方才有甜意。

    在东柵，懒懒散散的阳光洒落下来，温暖了世界。 早起的人家在水边画出一天里最早的音符，揭幕一天的序曲。石板小巷，小桥流水，水面偶尔荡起涟漪，水岸两边的民房倒影在水中，就这样安静的睡着，仿佛不想醒来。小巷里，偶尔有人与你相遇，又匆匆离开。

乌镇东栅景区

乌镇东栅景区


乌镇的东柵和西栅一样以河成街，街桥相连，依河筑屋，水镇一体。水阁、桥梁、石板巷等体现了中国古典民居“以和为美”的人文思想，体现了自然环境和人文环境相互交融的谐相美，呈现了江南水乡的古镇魅力。
但与商业化的西栅相比，东柵更完好的保存了许多古老的朴实气息，居民的电动车随意停放在自家门口，老狗慵懒地趴在巷口眯着眼打着哈欠，老妪抱脸盆来到河边洗衣服。。。黑瓦房，厚厚的青苔，被时光斑驳的墙壁，潺潺流水。。。

日升月落，静静生活。雨后初晴，静静生活。炊烟袅袅，静静生活。那些匆匆而来匆匆离去的旅游者们又能感觉到多少呢。
在这一刻。闭上眼睛，静静呼吸。
让宁静隔离一切浮华。
乌篷船悠闲地划过 ，泛起涟漪。
一切都是那么的静。

乌镇东栅景区

 
                                                            { 染坊，高杆船}来到东栅的宏源泰染坊，仿佛看到宋元年间的青花飘荡，天上是朵朵流云的晴空，地上却是自然飘逸的印花蓝布，在这用兰草染就的世界里，感受到的是自然淳朴的美丽。不远处，高杆船上直刺云霄的竹竿，只有勇者才能谈笑风生，就如钱塘江里的弄潮儿那般马踏平川。

宏源泰染坊




      “幽静长廊美人靠，碧水芊芊绕。枕水人家倚小桥，画船轻轻摇。”若是坐在乌镇的长廊里，再要一壶当年明清的龙井，也许可以品出当年细雨中春天的脚步。每天轮回的长廊的影子，就如钟表的时针，一天一天的赶着日子，计算着下场春雨到来的日子。但是，就在这古朴雅致，缠绵多情的江南，我似乎又看见数百年前，有一位神情幽怨的女子，迈着三寸金莲，默默地倚在临水的美人靠上，无声地停泊在渐渐深浓的暮色里，说不尽的凄婉动人。她在绣着比翼鸳鸯，等待心中的情郎，还是思念远方的亲人，或着又一个林妹妹正用心构思她的美文佳作，抑或是像古时大多数悲凄女子一样在叹息不公命运，将自己从一座宅院深深关进另一座深深宅院。

 
                                                                     { 逢源双桥 }
 镇上，有一座逢源双桥。这桥，左边有一座，右边有一座，左右逢源。不过，其实谁也不能左右逢源，中间隔着一个栏杆，你只能选择从一边走过来，不是左就是右。桥下有水栅栏，系古时水路进出关卡。传说踏走双桥有男左女右的习俗，走一遍桥，须分走左右两半，因此就演绎出走此桥便可左右逢源之说。当地人说，逢年过节，单身的男女，男左女右走逢源双桥，走过桥去，遇到的那个人便是跟定你一辈子的人。然后两个人牵着手一一走过镇上大大小小的七座桥，便能长久相伴一生。这样美丽的传说，给我们以莫大的安慰。坐在旁边的长廊上，渐渐的游人多了起来，听着不同导游的解说，便觉得可笑。有说“左升官，右发财”，多数人走了右桥，这只是应了世俗的愿望；有些人很贪心，他们想左右兼得，走了右桥又走左桥，殊不知据说，走逢源双桥是不可以回头的，一回头便又回到了原点。世事中鱼和熊掌总难以兼得，所以淡泊总归好些，不为所累，可静归真，是吾辈向往之。 讨厌美丽纯真的传说也要被旅游公司沾染上“升官发财”的味道。我愿化身石桥，受五百年风吹，五百年日晒，五百年雨打，只求我爱的此人从桥上走过。

逢源双桥

逢源双桥

                              
        站在逢源双桥上，是眺望乌镇美景之——财神湾的极佳视点，极目一望，东市河南岸垂柳依依，北岸水阁逶迤，令人心旷神怡。快傍晚的时候，我们守着逢源双桥，人渐渐少了，只剩我们。默默，无语。坐在桥上，望着水乡乌镇，开始有点点灯光。我们守着，一阵阵微风，些许凉意。其实乌镇的黄昏也很美，站在桥上，看看黄昏，看看落日。 夕阳西下，洒满落日余晖的古朴石桥或圆或弧,高低错落。此时此刻，在这么安静的地方，我真想就这样坐在河边无所事事的浪费时间，什么事都不去想，搬一张躺椅，喝茶、发呆，看桥下船来船往。。。
       乌镇的潺潺流水从远古流淌到今天，并且流向未来。只是现在的乌镇，也许成了一个水乡的标本，被人参观被人保护。可是这里等游人散尽了还是那么安静那么质朴，就像茅盾先生写的文字，大家都在这一河里洗漱、洗衣、洗马桶、汲水煮茶，也许外乡人看来觉得胃都要抽搐了，但是这就是乌镇，那个与水相关的乌镇。这里时间不曾流过，也不曾被改变什么！ 远离纷乱的都市，停下脚步时放开手送走烦恼，光影里的小桥流水人家，满载的是生活里的饱满的笑容，这就是生活在梦里的乌镇！用“似水年华”与乌镇匹配，确是相得益彰的。只是轻启唇齿悠悠地念出这四个字，脑海中已然浮现明明晦晦的年华似水，平静却不容挽留地向前流逝，波澜不兴。就像故事里黄磊无法挽留刘若英的离去，故事外的我们也无法挽留时光的脚步。

财神湾

财神湾



                                                                    很多到东柵的游客，都是为了追忆《似水年华》。有些人注定相遇、相爱却无法相守，如果一开始我们就能看到这段感情的终局，是否还会义无反顾地付出和相爱呢？故事里，黄磊与刘若英在东柵，隔着书架对望的那一眼，发酵了心底的爱情。 初时，相见相知，美好而淡然，回忆的时候嘴角不自觉地扬起微笑。回忆是否有保质期？一份初见时相遇相知的美丽回忆能够保存多少年？我不知道。但我想用力记住它们，记住乌镇。只有这样，当我日后转身回望的时候，才能觉得内心充盈，不觉得遗憾。



这个古风犹存的江南小镇,有着大气，自然，和谐之美。来到这里完全没有身处他乡之感。真是梦里不知身是客,错把他乡当故乡。
但我终是要回去的。
乌镇，离开。我感受了清晨小镇的静谧，爱上了夜晚的浪漫。我们带着好奇、喜悦而来，带着依依不舍眷恋的离开。偶尔回忆走过的地方，留下的时光的线条；你的世界，但愿都好；当我想起，你的静谧，你的浪漫，无意重读的那些记忆；时光悠悠，青春渐老，我们也曾迷失在似水年华里。

乌篷船又从眼前静静划过，划开河水一片片涟漪。
乌镇，许我一段似水流年
 乌镇，离开, 再见! 

                   
想附上刘若英在乌镇的一首广告词，觉得写得极好：

在这里安静地坐一会儿如何。
听船儿激荡起流水的细细声。
看着这里的人们是如何安详地生活.
离开纷乱的都市我来到这里 停下脚步
宁静可以让伤感隔离
时间真的不曾改变什么
放开手送走烦恼
光影里的小桥流水人家
满载的是生活里饱满的笑容
时间改变了许多事物
却不曾改变这里
那个笑的像花一样的孩子
一个轻快跳舞的女子
还有我的赤子之心
生活在梦里的乌镇
乌镇的一切都让我那么痴迷
它的黄昏
那晃晃悠悠的流水
那木阁楼上的文竹吊兰
那带着淡淡香气的蓝印花布
斑驳的青石板
温宛的爱情...
————刘若英
