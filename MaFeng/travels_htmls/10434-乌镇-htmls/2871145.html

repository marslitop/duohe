奇葩姐妹组的休闲之旅 西塘-乌镇

相关目的地：乌镇
去看看> 不再显示
  • 出发时间/2013-09-30
  • 出行天数/7 天
这篇游记豪迈地提到了：五月天客栈，西塘，西栅，东栅，猫的天空之城，管老太，塘东街，老木头酒吧，木锤酥

本来没打算来西塘和乌镇，在之前就一直是准备去厦门的，一直觉得厦门是最适合闺蜜旅行的地方，安静舒适，但说国庆出游鼓浪屿岛上肯定人很多，看了看鼓浪屿上的房价和机票，便宜的都给订完了，算了算预算差不多要4000-5000左右，觉得国庆出游不太划算，有想找个安静悠闲一点的地方，那就去西塘和乌镇吧。   （前奏有点啰嗦，屌丝气质立马浮现。第一次写游记见谅吧！！）                                              去西塘和乌镇之前在马蜂窝上查阅了很多篇游记，交通住宿之类的。以便于我们之后的出行非常顺利。由于是国庆，加上我们之前订房时间都比较晚了，临河的房间都给订完了。要不就是贵到离谱。是在没办法我们就住到了西塘景区外面和西栅景区的外面。        
前期准备：
      1、火车票                                                                                                                                              长沙-嘉兴 K135次列车，18:36开-----第二天早上7点到嘉兴（长沙是起点站，上海南是终点站）

     嘉兴-长沙  K137次列车，17；44开------第二天早上7点到长沙（上海南是起点站，长沙是终点站）

2、住宿
    前面也说了国庆的缘故人多，价格上涨没有住在景区里面，就住在景区外面。
我们住的是西塘一生一世客栈，梦雅筑房。（当时是偶然看到团购可以用的，99元加上国庆使用费230元，价格是328一天。从房间的图片上来看还是不错，就定了下来）


乌镇住宿订的是五月天客栈在艺龙订的房间。价格要348一天。（在西栅南门景区外面。大床房的价格和双床房的价格是一样的。订的是双床房。有一条可爱的38线。哈哈哈）

乌镇墅美Times时光酒店

3  门票
   西塘门票   100元/人   当地人代买50元/人
   乌镇    乌镇分东栅和西栅，东栅是100元一张，西栅120 元/人 联票是150元/人，西栅夜景是80元/人
10月1日第一天
       早上7点就到了嘉兴市火车站，到了火车站第一件事就是换回程的火车票，由于是国庆第一天那个取票的是人山人海。（ps：嘉兴市火车站是小站，没有自动取票机，长沙火车站有自助取票机很方便，我们9月30日是在长沙的代售点取票的,省去火车站排队的麻烦）
        由于嘉兴市火车站没有去西塘景区的直达车，只能在火车站前坪乘坐22路公家车到嘉兴市汽车北站。到达嘉兴市汽车北站不需要购买车票进车站里面有直达西塘的K152公交车。7元一人，大约四十分钟的样子到达西塘。我本以为是一人一座的旅游巴士，哪知道是公交车，我们两个只有一个人做了位置，那个车上真是挤啊，一辆车要挤100个人。还不是自动投币的，还有售票员在人群中穿来穿去的。
挤到我的五脏六腑都要破碎了。（PS：售票员超凶的。）
我们在汽车北站的五芳斋吃的早餐。一碗老北京杂酱拌面，一碗红薯粉汤。我可以说难吃吗？红薯粉汤就是白开水冲盐的味道。加点辣椒就是辣椒和盐味冲白开水的味道。炸酱拌面味道很一般，至少比难吃那个级别要上升一点。

五芳斋(恒泽堂北)

五芳斋(恒泽堂北)

 吃过早饭后，我们就坐上了去西塘景区的班车。到了西塘的之后，立马打电话给住宿的老板。老板刚好就在门口候着。把我们的行李放上车，就载我们回客栈了。路上要我们在他的小本子上面找自己入住的姓名和电话，我说没有找到，我说住三天的，4号早上走。老板直说知道知道，就只有你们两个人是住三天的，别人都是住两天就走的。老板人很好，很热情，虽然不是临河但是至少住着也很舒心。在客栈整理好之后就直接前往了西塘景区。
客栈离景区超近步行两分钟就到景区里面了，我们是从塘东街旁边的侧门进入景区的.在进景区的门口看到了管老太的臭豆腐,很多人排队,貌似挺有名的.管不得正不正宗,买一份在说.5元一份,一份五片

管老太臭豆腐(官方店)

外焦里嫩,味道还不错,辣椒酱不辣,挺咸的.还是比较还念长沙的西施臭豆腐.进入景区才知道那个人叫一个多啊.小桥上面全是人脑袋.先拍照只听见警察叔叔放着喇叭:人多拥挤,请勿在桥上逗留拍照,避免拥堵.
景区里面真的挺漂亮的,没有浓重的商业气息.水墨江南说的应该就是这样子了吧.

这张照片是在烟雨长廊上拍摄的,对岸的人喝着茶看着小桥流水好惬意,很多游客都在景区游船.鸬鹚站在小船上拍打这身体.看到这些景象整个人立马放松下来了.

 下面这张是在永宁桥上拍摄的,

沿线一路很多小店子,有很多可爱的又有趣商品,价格大多不贵.吃了一份粽叶包粉蒸肉.真心不是我的菜,但觉里面的肉米还没有熟.但是肉熟了,米吃起来是生的感觉,可能是国庆人太多了,生意好了做不赢的原因吧.
5元/份.再买来一份芒果布丁,味道挺棒的.10元一份,芒果味浓郁的,我们只走到了送子来凤桥就从前面的小桥去对岸了,话说对岸的街道很窄,被挤到头晕脑胀就找了一家店子吃甜品.一份花式刨冰,一份摩卡,两份加起来60元.小店买的猪脯肉是想现烤的,很大一片,10元一份.一份10片.肉很扎实,味道很不错,外面焦焦的,里面的肉还是嫩嫩的.还是热乎乎的.就是不能多吃.吃多了会腻的.
卡

 招牌奶茶，里面有点点果仁，味道很一般，价格挺高的。20元一杯。
 不知道是吃多了就发饭晕,还是因为在火车上睡了一晚没有休息好加上奔波了一天,人好困,不到六点两个人就回客栈休息了,不睡倒好,睡就到十点多了,爬起来又饿了.就出门去塘东街看看有没有好吃的地方.（ps：塘东街是西塘有名的酒吧一条街，真的是酒吧，低音炮开到超大，根本就没有清吧）我们想找个清吧都没有。 塘东街晚上人也超多的，经过大大小小的酒吧门口都有在拉客。我们已经过了泡吧的那个年纪了，泡不动了，只想找个清吧坐坐，可惜没有。无耐两个人打回道找了接口的一家烧烤店吃了晚餐加宵夜。

塘东街

 这家店老板是益阳人，也算上是个老乡了，很可惜，鱿鱼居然是水发的，吃起来有点腥味。总体味道还算不错了，看着隔壁桌吃小龙虾，掉口水了有木有。赶忙加一份小龙虾。真便宜，才38一份，不过个头也是挺小一个的。虽然个头小，味道做的很入味。一下子就消灭干净了。吃饱喝足之后就回客栈休息了。
 10月2日 第二天
       既然是休闲游那么肯定是要睡到自然醒的，一醒来已经是中午一点了，出门看到客栈老板在是类似于KFC之类的食品，看着都饿了，问了地方就跑过去吃了。实在是太饿了，都忙忘记拍照了，两个汉堡，一份薯条，一杯可乐，两对鸡翅一共才30元。好便宜啊。味道实在不必KFC差啊，重要的是分量也很足的。
吃饱了，之后在西塘里逛逛不知道去哪里好，就去给鱼儿喂了喂食。价格挺便宜的。两个人才50元、半小时。鱼儿咬脚脚好痒，但是好好玩。

沿途有许多特色小店。某个人情有独钟这家胡桃夹子的店子。

 很多特色小店，逛逛小店子时间一下就过去了。
拍了很多自拍照，最后发现抓拍才是我们的戏路。我们真是奇葩姐妹组。。。。

 
最可惜的是回来才发现只有大头照，没有合照。。。郁闷死了。两傻子。头饰是我们两在一店子买到的，只敢再外地带，回来只有放好珍藏的份了。挺贵的，15元一对。
我们两都是不爱吃正餐的。就爱吃吃小吃之类的。

下午四点的西塘，华灯初上，美不胜收啊。
 

 
傍晚的西塘更美，美中不足就是人太多了。烟雨长廊上差点都被挤到掉河里。
两人都想吃点辣的，景区外面看到有家重庆麻辣烫。

味道不太正宗。不过能吃到辣椒我真心欢喜啊。
       昨天在塘东街看到了一家很有特色的咖啡店。吃了逛了逛就去咖啡店了。店里可以写明信片，也可以代寄明信片，可以写定时明信片，几年后、十几年后，前提希望这家店子不要被拆除。有几只小猫，小猫咪很懒，不怕人，一直在睡觉，爬的太高了拍不到。味i道其实很一般。价格也算比较贵的了。丝袜奶茶加蛋糕40，拿铁28一杯。

呆了很长时间。才发现时间很晚了，该回客栈休息了。
10月3日  第三天。
     由于两个人都吃了感冒药，昏睡了一天，到了快晚上了才起来。说实话，生病去了旅游真心不舒服，还好出门前备好了感冒药和消炎药。不然，两个人就病倒在了客栈。
第一件事不用说都知道肯定是吃了，下图据说是当地特产木锤酥。

 鲜肉馄饨

 酒酿小丸子

 野菜大馄饨，野菜就是芹菜和大白菜保鲜肉。

馄饨汤特别鲜，后面才发现放了小虾仁。酒酿小丸子很香上面洒了桂花。
西塘夜景
安宁舒适。
 

买了四个河灯，放的匆忙，忘记许愿了。
今天要回客栈早点休息，明天一大早就要前往乌镇。
10月4日   第四天
   起了个大早，昨天晚上联系好了去乌镇，包的车230一辆，省去了中转的麻烦，西塘去乌镇没有直达的班车，要去嘉兴市中转，而且要转三次车才能到乌镇，我们觉得太麻烦了。干脆就包了车，价格挺合适的。到了乌镇和客栈的老板聊起来才知道我们谈的价格很便宜了，一般价格都是在260元左右了。一路上顺顺利利的。只可惜九点从西塘出发，接近中午才到乌镇的就客栈客栈休息了。
   在接我们去客栈的路上，客栈老板建议我们今天不要去景区了，一是人多，二是时间太晚了，他说要逛完景区时间不够，三是要我们去买景区的联票，不然不划算，要我们就在景区外面走走看看。
  到了北栅很小的一条街，客栈老板很热情告诉我们注意事项，走到不认识的路要我给立马给他打电话，
北栅我觉得是可以忽视的一条街道。两个人逛逛觉得饿了就去吃饭，想着来这边第四天了都没有吃过一顿米饭，都是吃小吃。
 上菜了。
 

 
 真心不是我们喜欢的口味，鱼很嫩但是鱼刺很多。蒸鱼豉油放多了有点咸。牛柳是甜的，我要吃辣的辣的，梭啰也是甜的，还没有入味，好奇怪的味道。但是油炸香芋很好吃。。。我不想说，虽然不好吃，但是我仍然咽下了两碗饭。我是真饿了。
乌镇除了景区，没有别处可以去的。只能早早的就回家了，客栈离景区有点距离，客栈老板很好，每天负责接送。
 
10月5日 第五天
    起了个大早，今天准备进景区了。买的联票，先逛的东栅。由于住西栅，我们就在西栅门口坐免费的巴士去东栅。巴士人上满就发车。十分钟到东栅。景区旅行团超多。到处都是人。船票是2人一船是200元。六人一船是360元。我们没有和别人拼船。（ps:我们是在西栅坐的船，而且是夜游的）
 

 
 
一进景区就有一大片竹林，路线进入景区  → 逢源双桥 → 财神湾  → 江南百床馆 → 江南民俗馆 → 高公生酒坊 → 宏源泰染坊 → 走仁义桥 → 传统手工作坊区→ 江南木雕馆 → 余榴樑钱币馆 → 茅盾故居、立志书院 → 古戏台 → 修真观 → 夏同善翰林第 → 皮影戏 → 传统商铺区 → 汇源当铺 →浙北分府(位于西栅景区边)

 乌镇东栅的景点。我们两是傻子，从立志书院就出来景区，应该直接往前走去古戏台 → 修真观 → 夏同善翰林第 → 皮影戏，应该看完之后再进景区的，我们直接绕过了这几个景点就进景区了。所以没看成。多傻啊。。。。好想哭。进景区是要刷票啊。景区设置只能刷两次票。 
东栅看点很少，没意思。而且，乌镇商业气息很浓重。没有西塘悠闲。 
从东栅出来就在门口坐去西栅的免费巴士。

10月6日 第六天

 今天就是准备回程了。中午十二点起床收拾行李准备回家了。老板把我送到乌镇汽车站。乌镇只有会嘉兴客运中心的班车。8.5元加一元保险费。大约四十分钟。走的那一天台风菲特要登陆了，乌镇也下起了大雨。在乌镇买回去嘉兴的车票。没有座位票了，只有站票。还好人不多。不太拥挤。到达客运中心。同一个停车场就有K19路公交车去嘉兴市火车站。半小时。2元一人。再火车站旁边的kfc坐着等火车的到来
 
 
这几天真是休闲自在，放空心情。行程很慢，所以人也不累。很开心的就过了这么多天了。
奇葩姐妹组的西塘之旅就到这。。。。。写出来怎么这么屌丝。。。哭
