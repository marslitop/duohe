自驾自贡-抚仙湖-洱海-大理天龙八部影视城-昆明亲子度假游

去看看> 不再显示
  • 出发时间/2019-08-02
  • 出行天数/7 天
  • 人物/家庭出游
  • 人均费用/2800RMB

前言：表妹和我们一家，本来打算两车自驾游，后来妹夫有事，少一个长途开车者。于是商定租一个7座别克GL8。原计划的环游，也缩短为 自贡 到 大理 ，然后原路返回。

此图为计划路线

实际路线如下：

d1 自贡 -抚仙湖
d2 抚仙湖- 昆明
d3 昆明 - 大理 -洱海才村
d4 洱海才村- 双廊
d5 洱海 双廊 古镇
d6 双廊 -天龙八部影视城- 昆明
d7 昆明 -回家

d1 自贡 -抚仙湖，汽车上的风光

早上八点20出发，两个小朋友，一路叽叽喳喳很开心。走到 宜宾 下了一阵雨，穿过几座山之后，天空开始变高，变蓝，太阳开始灿烂。 宜宾 到 昭通 一座山叠一座山，每座山都有一个圆圆的顶，布满绿植。因在 昭通 长宁 大桥有车祸，堵了一小时车，到悦乐服务区都快一点了，服务区自助中餐30元/人，菜色丰富味道不错。

图片：青山叠翠

吃完午饭，继续前进。过了 昭通 市以后，高速路面就变宽阔了，天空就像蓝色的底面上悬垂着各种形状的白色云朵，层次分明，漂亮极了，就像小时候家乡能看到的云朵。

图片： 昭通

高速路边，有个地图上看起来叫千塘子的地方很漂亮，水草拥着大大小小的湖泊，可惜不能靠边欣赏。

图片：千塘子

昭通 到抚仙湖的路，都比较平坦，和我们想象中的大山很不一样。我想 云南 这座春城，就算是为了这片天空，都值得一去。

图片：快到 昆明 时的云彩

晚上7：20到达抚仙湖月亮湾，找了一个抖音上好评超高的某家酒店入住。标间400元/间，比携程上还便宜，这个酒店不咋滴，连床上两枕头，都只更换了一个。24小时内停车费10元。找好住宿，去酒店门口，找了一家铜锅鱼吃，点了一个小锅铜锅青鱼，一个银鱼炒蛋，一个凉拌茄子，一个铜锅洋芋饭，味道还不错，晚饭总价245元。


d2 美丽又多变的抚仙湖

早上酒店附近吃早饭，10元/碗的磨肉面，吃完就去逛对面月亮湾湿地公园。

抚仙湖全图

抚仙湖开发得最成熟的景区是 禄充 ，住宿条件相对好点的是月亮湾，住宿条件最好的自然是希尔顿。景点集中在 禄充 -波息湾-希尔顿这一带，但是抚仙湖沿线，都是小渔村，镇子，很多民宿，住宿价格各有千秋，各取所需。
月亮湾湿地公园，入住酒店时给了一张卡可免费进入（门票20元/人，小朋友不买票）。公园里有一个人造湖，可以玩手摇船（儿童手摇船20元/20分钟，儿童独坐；亲子手摇船30元/20分钟，家长带小朋友；皮划艇价格没注意），皮划艇，人造沙滩里沙子细腻适合小朋友玩。

图片：月亮湾公园网红点

图片：月亮湾公园网红点

图片：月亮湾公园网红点

图片：月亮湾沙滩

月亮湾公园沙滩可赤脚玩耍，自行携带沙滩玩具，远远看去，是不是有海边的感觉？
沙滩凉棚20元/小时，以后每增加一小时加10元，赠两小瓶矿泉水，算得是良心好沙滩了 。
沙滩的前面是一片小石子，一直延伸至湖水，湖 水里 有许多小石头。玩水需要携带凉拖鞋，湖 水里 还有许多小鱼可捕捉，捕小鱼需自行携带小鱼网。玩沙累了，还可以乘脚踏船玩水，很适合孩子玩耍，整个抚仙湖因为整治，只有 禄充 才允许游泳，其他地方都是禁止游泳的。

图片：月亮湾湖边

中午就在月亮湾吃饭，尝试了一种红色的野生菌，味道还行。饭店门口彩色的马尾草很漂亮哟。店家门口15厘米左右高的石榴，居然结了好几颗鹌鹑蛋大小的石榴，我们很是惊奇的围观了一阵。

吃完午饭前往 禄充 风景区，门票15元/人，孩子没要票，停车费20元。

图片： 禄充 湖面（选滴船最少处 ）

禄充 建筑繁多，酒店、饭店林立。沙滩上人潮涌动，湖面上脚踏船密密麻麻，好热闹。沙滩上稀稀疏疏的小石子，需穿鞋子行走。带好泳衣，准备游泳，阳光明媚，嗯，准确的说是有些晒，探了下水温，然而水温很冰，我们放弃游泳，但是仍然有少许人在游泳。踩在浅水边的沙子上，陷脚……不便玩水，于是我们准备去玩脚踏船。刚刚开好票（脚踏船30元/人/时，1.1m以下儿童不要票），正准备选船，暴雨突然而至，岸上所有游客躲在开票的亭子里，亭子挤得密不透雨，远远望去湖面上的船拼命往岸边划来……

图片：第二场大雨后的抚仙湖

十多分钟后，大雨过去，阳光从云层透出来。我们改好出船时间，穿好救生衣，登上船，高高兴兴往湖中去……开出七、八米距离，发现浪很大，空心的船栏，在颠簸中，让我们很心惊 。不远处一条10来人的船，两个汉子站在船尾，掉进 水里 了（也不知道是不是故意的），很快他们又爬上船了 。看着远处巡逻的快艇，心又有了着落。

图片：笔架山（这山看起来就像座坟 ，据说山上的亭子看抚仙湖，那是格外的美呀）

在船上费力的蹬了半小时，两个小朋友就闹着要上岸（因为一直要求他们逮稳扶手，显得很无聊），我们选择了上岸。刚刚上岸，暴雨又来，我们又挤进亭子躲雨 。

图片：临走时，乌云盖住了对岸的山峦

雨停后，小朋友们玩起了堆沙游戏。两场大雨之后，湖面上的船少了一大半😨。玩了一段时间之后，小朋友们不愿意明天爬山，于是决定返回 昆明 。刚踏上旅途，大雨又至。
抚仙湖还有孤岛、波息湾、界鱼石等景点，也有浮潜基地，美食抗浪鱼等。因着抚仙湖尸库和水底古城的名声，我们心里还是有点疙瘩滴😥。更适合小朋友的景点是月亮湾，景色更好的是 禄充 。

d3 昆明 - 大理

头晚在时代广场小龙坎吃完火锅之后，去花千谷酒店住宿，因为新开张的酒店房间味道较大，然后选酒店选到晚上10点才入住。
早上吃完丰盛的早饭，从 昆明 出发到 大理 ，出发时间9点30 。

图片：高速路边的烟叶苗

杭瑞高速大红田隧道附近修路，到 大理 就3点了。一路上大片大片的烟叶苗长得很茂盛（洱海环海路才村至海舌公园附近还有红塔集团烟叶基地）， 云南 的烟草公司是我国的纳税大户🤔。

洱海地图

导航到 大理 古城后，驱车到才村，发现到不了电话预订的酒店看房（其实车是可以到的，因为洱海整治，那条路才拆出来的，还是泥胚，地图上没有 ）。然后寻找了两个小时住宿，最后太累了，随便住了一个民宿😒。 大理 古城和天龙八部影视城也没去看。

图片：在才村码头前面沟渠里洗菜的村民

才村码头车流量大，很多旅行大巴，经常堵车，才村公路很多窄巷道，开车看见小巷道、机耕道慎入，我们的车就在这里亲密的和矮桩接触了 。

图片：才村码头

晚饭在住宿店家推荐的榴恋园，吃了油炸豌豆腐、黄焖鸡还不错，其他都是点的川菜。码头可以看日落，然而小雨，运气差了点。

d4 才村坐船游罗荃半岛-浅游 双廊

早上吃完饭，才村码头买票坐船，一开始以为是环海游……后来才知道，就是直线到对面的罗荃半岛，往返加罗荃半岛门票180元/人，1.2米以上儿童半票。

图片：远处的两塔处就是罗荃半岛

岛上有马帮文化，两塔，观音阁，高山漂流等景点，上罗荃塔需10元/人，乘电梯另收费。环岛游可以不坐船进这个景点30元/人，1.2m以下孩子免费。

图片：散客所坐游艇

10点坐完游艇，天空开始下雨，雨渐渐变大。因为连续两天都在找住宿的路上花费大把时间，所以舍去了 喜洲 古镇和海舌公园。海舌公园舌尖部分是网红景点，能拍出大片的感觉，这个公园是不要门票的，停车费10元，在附近饭店吃饭可免停车费。
我们直接开车到看中的 双廊 临海 客栈，选中卧榻之后再吃午饭（去的时候正在下雨，全景海景房和老板讲价，直接比携程便宜了400元/间，因为下雨海景房可能用处不大）。

图片：卧榻😰（妹子说终于和想象中的洱海有点像了）

图片：客栈看夕阳

其实洱海环线各类住宿是很多的，这次的旅途，在暑假旺季，要找心仪的住宿需要提前预订。因为各种原因俺们找住宿花了很多时间 。所以观景路线都乱了……
午饭后，太阳开始发力，孩子们就去水边玩水去了……

图片：波光粼粼的湖水

这一玩就4点了，收拾一番，逛这商业繁华的渔村。一路逛、吃……
