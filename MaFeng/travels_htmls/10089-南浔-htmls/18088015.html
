值得一去的江浙沪皖古镇集锦

还没有添加游记头图
去看看> 不再显示

中国 上下五千年，尤其古镇文化在江浙皖地区发展的淋漓尽致，让我们一同去看看江浙皖地区值得去的一些古镇吧。
/ 无锡 荡口古镇 /
走进了古镇，迎接你的就是一条小河，两岸白墙黑瓦，大红灯笼高高挂，河面上飘着几叶小舟。

/ 苏州 锦溪 古镇 /
锦溪 现存民居大部分是明清和民国时建造的，居民临水而建，房屋多砖木结构，临水开门窗或建筑水埠码头，古色古香。

/ 上海 朱家角 古镇 /
浓浓的文艺范儿，不仅是艺术演出和展览，还能在街头巷尾那一家家装饰得极富创意的小咖啡馆、酒吧以及民宿里找到。

/ 浙江 绍兴 安昌 古镇 /
安昌 可以说是 绍兴 最原汁原味的水乡古镇了，几乎没有商业开发，在古朴的老街民居中走走，感受古镇那质朴的生活气息。

/ 浙江 湖州 南浔 古镇 /
自明代 南浔 就是 江南 雄镇，如今历史隐去，清晨浣洗的人们，映水的 朝阳 ，炊烟袅袅，这里的生活仍旧是原来的模样。

/ 安徽 黟县 南屏 古镇 /
建于元明，至今保存着8幢古祠堂，300多幢明清古建筑，沿着72条巷弄分布如迷宫似的飘荡。《菊豆》和《卧虎藏龙》都在此取景。

/ 安徽 宣城 查济 古镇 /
漫步在 查济 小路上，仿佛走进水墨画作一般。白墙黑瓦的徽式建筑，质朴的村民，这里的祠堂、这里的名人故事还在坊间流传着。

/ 安徽 黄山 呈坎 古镇 /
青山绿水白墙黛瓦，拥有1800多年的历史，是世界迄今发现最古老、保存最完整的八卦风水古村落。

/ 浙江 嘉兴 乌镇 /
游玩建议：东栅较小未过多开发，也比较原始，建议白天游玩1.5小时；西栅开发成熟，游玩点较多，且夜景很美。

/ 浙江 嘉兴 西塘 古镇 /
瘦瘦清清的小河，高高低低的石桥，印象中 西塘 总给人舒服的感觉。

/ 江苏 苏州 木渎 古镇/
江南 著名的风景名胜区，素有“吴中第一镇”、“秀绝冠 江南 ”之誉。 木渎 古镇是与 苏州 城同龄的汉族水乡文化古镇，已有2500多年历史。

/ 江苏 苏州 沙溪 古镇/
沙溪 古镇位于 江苏 省 苏州 市 太仓 市，境内有明清临水建筑、古巷、古桥和名人故居等。

/ 江苏 苏州 千灯 古镇/
千灯 古镇距今已有2500多年的历史，是爱国学者顾炎武的故乡，又是昆曲的发源地。 千灯 原名“千墩”，名出吴越争霸。

/ 江苏 苏州 甪直 古镇/
甪直 古镇是具有2500年历史的 江南 水乡古镇，水多，桥多，巷多，古宅多，名人多。主要的景点有保圣寺、叶圣陶纪念馆、 江南 文化园等。

/ 上海 金山 枫泾古镇/
枫泾镇是 中国 历史文化名镇，亦为新沪上八景之一，历史上因地处吴越交汇之处，素有吴越名镇之称；枫泾为典型的 江南 水乡古镇。

/ 江苏 昆山 周庄 古镇/
“ 中国 第一水乡”，是国家首批5A级景区。 周庄 始建于1086年（北宋元祐元年），因邑人周迪功先生捐地修全福寺而得名。春秋时为吴王少子摇的封地，名为 贞丰 里。现在商业气息有些重。

/ 江苏 泰州 溱潼古镇 /
泰州 溱潼古镇 位于 江苏 省 泰州 市姜堰区溱潼镇，该镇四面环水，环境优美，为鱼米之乡。

/ 江苏 苏州 同里 古镇/
同里 古镇属于 江苏 省 苏州 市吴江区，宋代建镇。镇区内始建于明清两代的花园、寺观、宅第和名人故居众多，“川”字形的15条小河把古镇区分隔成七个小岛，而49座古桥又将其连成一体，以“小桥、流水、人家”著称。
